🎯 SYSTEM OVERVIEW
═══════════════════════════════════════════════════════════════════════════

This is a complete research platform for morphogenetic soft robotics:

  Frontend (Next.js):
    • Dashboard with parameter controls
    • 3D mesh visualization (Three.js)
    • Experiment tracking & comparison
    • Research documentation

  Backend (Python):
    • Morphogen PDE solver (JAX)
    • Soft body physics simulator (FEM)
    • Neural controller (Lyapunov + SNN)
    • Optimization engine

  API:
    • 6 RESTful endpoints
    • Real-time simulation
    • Experiment logging

🛠️  COMMON COMMANDS
═══════════════════════════════════════════════════════════════════════════

Development:
  $ pnpm dev                      # Frontend dev server
  $ cd python && python app.py    # Backend server

Production:
  $ pnpm build                    # Build frontend
  $ pnpm start                    # Start production server
  $ FLASK_ENV=production python app.py    # Production backend

Docker:
  $ docker-compose up --build     # Full stack in Docker

Testing:
  $ pnpm test                     # Frontend tests
  $ pytest                        # Backend tests
