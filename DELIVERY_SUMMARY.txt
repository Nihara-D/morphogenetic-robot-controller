================================================================================
                         RUN COMMANDS SUMMARY
================================================================================

LOCAL DEVELOPMENT (Recommended):

Terminal 1 - Frontend:
  $ cd morphogenetic-robot-controller
  $ pnpm dev
  → http://localhost:3000

Terminal 2 - Backend:
  $ cd morphogenetic-robot-controller/python
  $ source venv/bin/activate      # macOS/Linux
  $ # or: venv\Scripts\activate   # Windows
  $ python app.py
  → http://localhost:5000


PRODUCTION BUILD:

Build:
  $ pnpm build
  $ pnpm start

Backend (separate terminal):
  $ cd python
  $ source venv/bin/activate
  $ FLASK_ENV=production python app.py


DOCKER (FULL STACK):

  $ docker-compose up --build


QUICK VERIFICATION:

1. Check versions:
   $ node --version    # Should be 18+
   $ python --version  # Should be 3.9+
   $ pnpm --version    # Should be 8+

2. Install dependencies:
   $ pnpm install
   $ cd python && pip install -r requirements.txt

3. Start servers (2 terminals)
4. Visit http://localhost:3000

================================================================================
                         API ENDPOINTS
================================================================================

All endpoints are through Next.js proxy to Python backend:

POST /api/simulation/morphogen
  └─ Run morphogen field (Gray-Scott PDE) simulation

POST /api/simulation/physics
  └─ Run soft body physics simulation

POST /api/simulation/run
  └─ Full end-to-end simulation (morphogen + physics)

POST /api/controller/optimize
  └─ Optimize controller parameters (evolutionary)

GET /api/experiments
  └─ List all saved experiments

See README.md for full cURL examples.
